# 🚀 财经新闻分析器 - ClawHub 发布指南

## 快速发布（3 步）

### 第 1 步：打开浏览器登录

**点击链接登录 ClawHub：**
```
https://clawhub.ai/cli/auth
```

1. 使用你的账号登录（没有就注册一个）
2. 完成验证

### 第 2 步：验证登录状态

打开 PowerShell，运行：
```powershell
clawhub whoami
```

如果显示你的用户名，说明登录成功！✅

### 第 3 步：执行发布命令

```powershell
cd C:\Users\Administrator\.openclaw\workspace\skills\finance-news-analyzer

clawhub publish . `
  --slug finance-news-analyzer `
  --name "财经新闻分析器" `
  --version 1.0.0 `
  --changelog "初始版本：支持 28+ 新闻源、10+ LLM 模型、情感分析、图表生成、数据持久化、定时任务"
```

## 常见问题

### Q: 提示 "Not logged in"
**A:** 先运行 `clawhub login`，在浏览器完成登录

### Q: 发布失败 - 网络错误
**A:** 检查网络连接，或稍后重试

### Q: Slug 已被占用
**A:** 换一个名字，比如 `finance-news-analyzer-pro`
