## 使用示例

### 安装
```bash
pip install pdf-ocr-skill
```

### 使用
```python
from pdf_ocr_processor import PDFOCRProcessor

processor = PDFOCRProcessor()

# 识别PDF
result = processor.ocr_pdf('document.pdf')

# 识别图片
result = processor.ocr_image_file('image.jpg')

print(result['text'])
```
