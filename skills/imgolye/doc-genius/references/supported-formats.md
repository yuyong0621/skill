# 支持的格式

## 输入格式

### 文档类型

| 格式 | 扩展名 | 提取方式 | 页数统计 |
|------|--------|---------|---------|
| PDF | .pdf | PyPDF2 | ✅ |
| Word | .docx, .doc | python-docx | ❌ |
| Markdown | .md, .markdown | markdown库 | ❌ |
| 纯文本 | .txt | 直接读取 | ❌ |

### PDF支持

- ✅ 标准PDF
- ✅ 扫描件（需要OCR，实验性）
- ⚠️ 加密PDF（需要密码）
- ⚠️ 图片PDF（需要OCR）

### Word支持

- ✅ .docx（Office 2007+）
- ⚠️ .doc（旧格式，部分支持）

---

## 输出格式

### JSON

```json
{
  "file": "document.pdf",
  "type": "pdf",
  "summary": "文档摘要...",
  "keywords": ["关键词1", "关键词2"],
  "word_count": 5000,
  "pages": 12
}
```

### Markdown

```markdown
# 文档摘要

**文件**: document.pdf  
**类型**: pdf  
**字数**: 5000  
**页数**: 12

---

## 摘要

文档摘要内容...

---

## 关键词

关键词1, 关键词2
```

### HTML

完整的HTML文档，包含样式。

| 输入 → 输出 | Markdown | HTML | Word | PDF |
|------------|----------|------|------|-----|

**图例：**
- ✅ 完全支持
- ⚠️ 实验性支持
- 🔜 计划中

## 性能限制

### 文件大小

| 格式 | 推荐大小 | 最大大小 |
|------|---------|---------|
| PDF | < 10MB | 50MB |
| Word | < 5MB | 20MB |
| Markdown | < 1MB | 5MB |
| 纯文本 | < 1MB | 5MB |

### 页数限制

- **推荐**：< 100页
- **最大**：500页
- **超大文档**：建议分块处理

---

## 编码支持

### 支持的编码

- ✅ UTF-8（推荐）
- ✅ UTF-16
- ⚠️ GBK（中文）
- ⚠️ Big5（繁体中文）

### 自动检测

系统会自动尝试检测编码，但建议使用UTF-8。

---

## 特殊格式

### 表格

- ✅ Word表格 → Markdown表格
- ⚠️ PDF表格 → 文本（格式可能丢失）
- ✅ Markdown表格 → HTML表格

### 图片

- ⚠️ PDF图片 → 不提取（仅文本）
- ⚠️ Word图片 → 不提取（仅文本）
- ❌ 图片OCR → 计划中

### 公式

- ❌ 数学公式 → 不支持
- ❌ 代码高亮 → 不支持

---

## 已知问题

### PDF

1. **扫描件**：需要OCR支持（实验性）
2. **复杂布局**：可能丢失格式
3. **多栏排版**：文本顺序可能错乱

### Word

1. **旧格式(.doc)**：部分功能不支持
2. **复杂样式**：可能丢失格式
3. **嵌入对象**：不提取

### Markdown

1. **自定义语法**：部分不支持
2. **HTML混合**：可能丢失部分内容

---

## 最佳实践

### 提高提取质量

1. **使用标准格式**
   - PDF：避免扫描件
   - Word：使用.docx格式
   - 文本：使用UTF-8编码

2. **优化文档结构**
   - 清晰的标题层级
   - 避免过于复杂的布局
   - 合理的段落分隔

3. **处理大文件**
   - 分块处理
   - 降低并发数
   - 增加超时时间
