# crypto-lending-optimizer - 加密借贷利率优化器

**痛点**：借贷平台太多难以选择，利率波动大，手动切换收益低，错过最优利率
**价值**：实时监控全平台利率，自动切换最优收益，年化提升 25-40%

## 使用方式

```bash
crypto-lending-optimizer --help
```

## 功能特点

- ⚡ **多平台利率对比** - 监控 Aave/Compound/JustLend 等 50+ 平台实时利率
- 📊 **风险调整收益** - 综合考虑利率、平台风险、流动性给出评分
- 💰 **自动复投策略** - 利息自动复投，最大化复合收益
- 🔒 **抵押率监控** - 实时预警清算风险，自动补充抵押物
- 🎯 **最优借贷方案** - 根据借款需求推荐最低成本方案

## 适用场景

- 稳定币理财收益最大化
- 加密资产抵押借贷
- DeFi 杠杆策略优化
- 机构级资金管理

## 配置

在 `~/.openclaw/workspace/config/crypto-lending-optimizer.json` 配置 SkillPay API Key。
