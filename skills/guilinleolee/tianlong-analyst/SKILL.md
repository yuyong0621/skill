# tianlong-analyst - 天龙需求分析专家

> 天龙引擎 V7.4 核心能力产品化 | 批判性思维评分：0.96

---

## 🎯 功能描述

基于天龙引擎 00 分析师的需求解构能力，帮助用户：
- 挖掘需求本质与问题根因
- 识别复杂度/类型/优先级
- 输出结构化分析报告

---

## 🔧 使用方式

```bash
# 基础用法
/tianlong-analyst <需求描述>

# 示例
/tianlong-analyst 我想做一个 AI 写作助手
```

---

## 📋 输出格式

### 1. 需求解构
- 表面需求 vs 真实需求
- 用户画像分析
- 使用场景识别

### 2. 复杂度评估
| 级别 | 判断标准 | 执行方式 |
|------|---------|---------|
| 🟢 绿灯 | 简单、明确、低风险 | 直接执行 |
| 🟡 黄灯 | 中等复杂、多方案 | 给 2-3 方案 |
| 🔴 红灯 | 高风险、高成本、战略 | 详细方案 + 决策 |

### 3. 批判性思维质疑
- 第一性原理分析
- 激励分析
- 二阶思维推演
- 非对称回报评估

### 4. 行动建议
- 立即执行项
- 需要决策项
- 风险预警项
