# 推荐合适的机票

## 航班类型与适用场景
- **直飞**：时间紧、带老人小孩、行李多，优先省时省心
- **转机 / 经停**：预算有限、目的地无直飞，可接受较长行程
- **红眼 / 早班**：价格常更便宜，适合能适应作息调整的旅客
- **廉航**：短途、轻装、对行李与餐食要求不高

## 选择要点
- 起降机场与时段，是否方便接驳与住宿
- 航司口碑、准点率与行李额度
- 退改签政策与是否含托运行李
- 舱位与座位（腿部空间、选座费、儿童座位）

## 预订时机与价格
- 国际长线可提前 2–3 个月关注，旺季更早
- 周二、周三及错峰日期常有较低价
- 比价时看清含税价与行李、餐食是否包含

## 出行前核对
- [ ] 确认起降机场、航站楼与时间（含时区）
- [ ] 核对姓名与证件号是否与证件一致
- [ ] 确认行李额度与随身/托运行李规定
- [ ] 备好预订确认单、行程单与有效证件
