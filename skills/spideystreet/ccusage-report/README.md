# Claude Code Usage Report

Report Claude Code token consumption and costs using ccusage.

## Requirements

- `bunx` (bun) — runs `ccusage` CLI
- Claude Code session data on the local machine

## Setup

No setup required. `ccusage` is fetched automatically via `bunx`.

## Usage

> "Show my claude code usage"

> "Combien j'ai dépensé cette semaine ?"

> "Monthly report with model breakdown"

## Install

```bash
clawhub install spideystreet/ccusage-report
```
