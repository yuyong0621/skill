# Portfolio Management - 持仓分析与投资建议

## 功能概述

每个工作日为用户提供两次持仓分析：
- **早盘分析（9:00）**：盘前研判，结合隔夜消息与技术信号，给出当日操作建议
- **盘后复盘（16:30）**：收盘总结，分析当日走势，给出次日策略；同时提醒用户上传最新持仓截图

## 用户画像

- **持仓市场**：A股、港股、ETF/基金
- **投资风格**：灵活切换，根据市场环境调整攻守节奏
- **分析深度**：深度分析（含详细技术指标、资金流向、财报解读、行业对比）

## 工作流程

### Step 1: 读取持仓截图

使用Read工具读取用户上传的持仓截图，提取以下信息：
- 股票/ETF代码与名称
- 持仓数量与成本价
- 当前市值与盈亏比例
- 持仓占比

如截图模糊或信息不全，主动询问用户补充。

### Step 2: 搜集最新市场信息

对每只持仓标的，使用WebSearch搜索以下信息：

**技术面：**
- 当前股价与近期K线走势
- 关键均线位置（MA5/MA10/MA20/MA60）
- MACD、KDJ、RSI等技术指标信号
- 成交量变化与量价关系
- 关键支撑位与压力位

**资金面：**
- 主力资金净流入/流出
- 北向资金（A股）/南向资金（港股）动向
- 融资融券余额变化
- 大宗交易数据
- 板块资金流向

**基本面：**
- 最新财务数据（营收、净利润、PE、PB、ROE）
- 近期公告与重大事项
- 行业政策与产业链动态
- 机构评级与目标价变动
- 所在行业景气度

**宏观环境（数据准确性强制要求）：**
- 当日/近期重要经济数据
- 央行货币政策动向
- 中美关系等地缘因素
- 市场整体情绪指标

**数据准确性保障机制（必读）：**

1. **大盘指数数据验证流程**
   - 必须搜索当日收盘数据（关键词："[日期] 上证指数 收盘 涨跌幅"）
   - 至少从2个独立来源验证（如新浪财经+东方财富）
   - 明确区分：盘中数据 vs 收盘数据 vs 历史数据
   - 必须标注数据来源和时间戳

2. **数据时效性检查清单**
   - [ ] 检查搜索结果的时间戳是否为当日
   - [ ] 排除3日以上历史数据干扰
   - [ ] 区分"午间"与"收盘"数据
   - [ ] 确认数据与报告日期一致

3. **常见错误防范**
   - ❌ 错误：将3月3日数据当作3月6日数据
   - ❌ 错误：将盘中-1.43%当作收盘数据
   - ❌ 错误：混用不同日期的数据
   - ✅ 正确：每个数据都标注日期和来源

4. **数据输出格式**
   ```
   | 指数 | 收盘点位 | 涨跌幅 | 数据来源 | 数据时间 |
   |------|---------|--------|---------|---------|
   | 上证指数 | XXXX.XX | +X.XX% | 新浪财经 | 2026-03-06 15:00 |
   ```

### Step 3: 逐只分析并给出建议

对每只持仓标的输出以下结构化分析：

```
### [股票代码] [股票名称]

**综合评级：** 强烈推荐持有 / 建议持有 / 观望 / 建议减仓 / 建议清仓

#### 技术面分析
- 趋势判断（上升/震荡/下降）
- 关键价位（支撑位、压力位）
- 技术指标信号（MACD金叉/死叉、KDJ超买/超卖等）
- 量价配合情况

#### 资金面分析
- 主力资金动向
- 北向/南向资金态度
- 融资融券信号
- 筹码集中度变化

#### 基本面分析
- 估值水平（PE/PB与行业对比）
- 业绩趋势（营收/净利润增速）
- 近期催化剂（公告、政策等）
- 行业竞争格局

#### 操作建议
- 明确建议：加仓 / 持有 / 减仓 / 清仓
- 目标仓位比例
- 建议操作价位区间
- 止盈位与止损位
```

### Step 4: 整体仓位建议

```
## 整体仓位建议

- 当前总仓位评估
- 建议总仓位比例（如：建议控制在XX%）
- 各标的建议仓位比例
- 行业集中度风险提示
- 整体攻防策略
```

### Step 5: 新标的推荐（强制要求）

**重要：** 每次盘后复盘必须包含新标的推荐，不得省略。这是技能的核心功能之一。

**触发条件：**
- 只要有减仓/清仓操作释放资金 → 立即推荐配置标的
- 即使没有操作 → 提供1-2只观察标的供后续参考
- 即使满仓 → 说明"当前满仓，建议减仓后配置以下标的"

**简化原则：**
- 时间充裕（>30分钟）：执行完整六步筛选流程
- 时间紧张（<15分钟）：使用简化流程，但至少验证3只候选标的
- 紧急情况：直接基于持仓缺陷给出最匹配的1-2只标的

**输出要求：**
- 至少推荐1只标的，最多3只
- 每只标的必须包含：推荐理由、介入价位、目标价、止损价、建议仓位
- 必须说明与现有持仓的互补性

#### 5.1 确定筛选方向

基于现有持仓分析，识别需要填补的空白：
- **行业空白**：现有持仓未覆盖的板块（如能源、公用事业、贵金属等）
- **风格空白**：高股息防御、低波动、避险资产等
- **风险分散**：与现有重仓板块负相关或低相关的标的

#### 5.2 构建候选池

使用WebSearch搜索以下信息构建候选池：

```
搜索关键词示例：
- "北向资金 连续净买入 个股 [日期]"
- "主力资金净流入 排名 [日期]"
- "ETF 资金净流入 排名 [日期]"
- "高股息 股票 推荐 [年份]"
- "机构金股 [年份] [月份]"
- "[行业] 景气度 [年份]"
```

候选标的数量：6-10只，覆盖不同行业/风格

#### 5.3 三维度打分（满分100分）

对每只候选标的进行量化评估：

| 维度 | 权重 | 评估指标 | 评分标准 |
|------|------|---------|---------|
| **技术面** | 30% | 趋势方向、均线排列、MACD/KDJ信号、量价配合、支撑压力位 | 多头趋势+均线多头排列+指标金叉=25-30分；震荡=15-24分；空头趋势=0-14分 |
| **资金面** | 35% | 主力净流入、北向/南向资金、融资融券、机构持仓变化 | 连续净流入+资金大幅流入=30-35分；分歧=15-29分；持续流出=0-14分 |
| **基本面** | 35% | PE/PB估值、业绩增速、行业景气度、催化剂、股息率 | 低估+高成长/高股息+催化剂=30-35分；合理=15-29分；高估+业绩下滑=0-14分 |

**评分等级：**
- 85-100分：强烈推荐
- 75-84分：推荐
- 60-74分：观望
- <60分：排除

#### 5.4 互补性评估

对高分标的进行互补性检查：
- **行业重叠**：是否与现有持仓同一行业？（权重-10分）
- **风格重叠**：是否与现有持仓同一风格？（权重-5分）
- **相关性**：是否与重仓标的正相关？（权重-10分）

#### 5.5 最终筛选

按综合得分排序，选择Top 2-3只：
- 优先选择得分最高且互补性强的标的
- 确保推荐标的覆盖不同行业/风格
- 单只标的建议仓位不超过5%

#### 5.6 输出格式

```
## 新标的推荐

### 推荐1: [代码] [名称] | 综合得分: [XX分]

**筛选逻辑：**
- 入选原因：[与现有持仓的互补性说明]
- 三维度得分：技术面[X/30] + 资金面[X/35] + 基本面[X/35] = [总分]

**推荐理由：**
- 技术面：[具体信号，如"站上所有均线，MACD金叉"]
- 资金面：[具体数据，如"近5日主力净流入XX亿，北向资金增持"]
- 基本面：[具体逻辑，如"PE仅X倍，低于行业均值30%，股息率X%"]

**操作建议：**
- 建议介入价位：XX - XX元
- 目标价：XX元（潜在收益 XX%）
- 止损价：XX元（最大亏损 XX%）
- 建议仓位：X%
- 风险提示：[主要风险点]

### 推荐2: ...
```

#### 5.7 简化流程（时间紧张时）

若时间有限，可跳过完整筛选，直接基于持仓缺陷给出最匹配的互补标的：
1. 识别最明显的1-2个持仓缺陷
2. 搜索对应方向的优质标的
3. 对候选标的进行简要三维度验证
4. 输出推荐

**注意：** 简化流程需至少对推荐标的进行单独搜索验证（技术面+资金面+基本面），避免纯主观推荐。

### Step 6: 盘后提醒（16:30触发）

在下午四点半的分析结尾附加提醒：

```
---
**提醒：** 请上传今日最新持仓截图，以便明日早盘为您提供最新分析。
```

## 注意事项

- 所有建议仅供参考，不构成投资建议
- 在报告末尾始终附加免责声明
- A股标的搜索时使用"沪深"或交易所前缀
- 港股标的搜索时使用港交所代码格式
- ETF标的需关注跟踪误差和折溢价率
- 数据以最新公开信息为准，标注数据时间
