# 炎月短期记忆使用指南

## 🎯 简单设计（星之君需求）

**核心功能**：
1. 记录别的通道每分钟说了什么
2. 最多记录30分钟
3. 超时整合到MEMORY.md
4. 每次会话 = 会话本身 + 原记忆 + 临时记忆

---

## 📋 使用流程

### 1. 星之君在飞书发消息
```
星之君（飞书）: "帮我记住要完成X任务"
```

**炎月自动执行**：
```python
ca.record("feishu", "星之君要求完成X任务")
```

### 2. 30分钟内，星之君在QQ问
```
星之君（QQ）: "我刚才说了什么？"
```

**炎月自动提供上下文**：
```
会话上下文 = 当前会话 + MEMORY.md + 临时记忆

[其他通道最近30分钟活动]
- [feishu] 14:30: 星之君要求完成X任务
```

### 3. 30分钟后，自动整合到MEMORY.md
```
临时记忆中的内容过期后，自动写入MEMORY.md作为永久记忆
```

---

## 🔧 使用示例

### 记录通道活动
```python
from channel_activity import ChannelActivity

ca = ChannelActivity()

# 飞书通道记录
ca.record("feishu", "星之君要求短期记忆30分钟TTL")

# QQ通道记录  
ca.record("qq", "测试QQ通道活动")
```

### 获取其他通道活动
```python
# 在QQ会话中，获取飞书的活动（排除当前通道）
summary = ca.get_context_summary(channel="qq")
print(summary)

# 输出：
# [其他通道最近30分钟活动]
# - [feishu] 15:00: 星之君要求短期记忆30分钟TTL
```

### 整合到会话
```
每次AI回复时：
1. 当前会话历史（系统自动提供）
2. MEMORY.md内容（长期记忆）
3. 通道活动摘要（临时记忆，使用get_context_summary）
```

## ⚙️ 核心特性

### ✅ 简单设计
- 只有核心功能，不复杂
- 记录、查询、清理

### ✅ 按分钟记录
- 每条记录带时间戳
- 自动清理超时内容

### ✅ 30分钟TTL
- 超时自动删除
- 可配置升级到MEMORY.md

### ✅ 多通道支持
- feishu、qq等通道独立记录
- 可排除当前通道

---

## 🎯 解决的问题

**原问题**：
- 飞书讨论的内容，QQ看不到
- 多通道上下文不连贯

**解决方案**：
```
会话上下文 = 当前会话 + MEMORY.md + 通道活动
```
