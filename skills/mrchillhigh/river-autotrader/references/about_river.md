# River 项目详解 / River Project Details

## 什么是 River? / What is River?

River 是一个基于 Omni-CDP 协议的链抽象稳定币系统，专注于比特币生态。

River is a chain-abstraction stablecoin system powered by the Omni-CDP protocol, focused on Bitcoin ecosystem.

## 核心产品 / Core Products

### 1. satUSD
- 链抽象稳定币
- 支持多条链: Ethereum, BNB, X Layer, Hemi, Arbitrum, Base, BOB, Sonic, B²
- 与美元 1:1 锚定

### 2. Smart Vault
- 智能金库
- 自动化收益策略

### 3. Prime Vault
- 尊享金库
- 高收益产品

### 4. Bridge
- 跨链桥
- 支持资产跨链转移

### 5. satUSD Earn
- 质押 satUSD 获得收益

### 6. $RIVER Token
- River 生态治理代币
- Staking 获得投票权和收益

## RIVER 代币经济 / RIVER Tokenomics

- **用途**: 治理、质押、收益分享
- **Staking**: 质押 RIVER 获得投票权
- **官网**: https://app.river.inc/river

## 风险提示 / Risk Warning

1. 加密货币投资风险高
2. 可能存在智能合约风险
3. 市场波动可能导致损失
4. 请仅投资可承受损失的资金
