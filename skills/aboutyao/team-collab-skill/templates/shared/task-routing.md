# 任务路由规则

## 产品关键词
- 规划、分析、需求、功能、产品
- 怎么做、帮我想想、有什么建议
- 优先级、方案、决策

## 研发关键词
- 安装、技能、代码、GitHub
- 技术实现、集成、API
- 配置、部署、自动化

## 运营关键词
- 小红书、发帖、文案、图片
- 数据、分析、监控
- 内容、创作、发布

## 示例

| 用户说 | 路由到 |
|--------|--------|
| "帮我规划一个功能" | 产品 |
| "装一个 GitHub 技能" | 研发 |
| "帮我写篇小红书文案" | 运营 |
| "搭建一个完整系统" | 产品 → 研发 → 运营 |
