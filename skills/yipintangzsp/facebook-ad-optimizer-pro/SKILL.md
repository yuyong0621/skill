# facebook-ad-optimizer-pro - Facebook 广告优化专业版

## 技能描述
专业版 Facebook 广告优化工具，深度分析广告表现，提供创意优化、受众调整、出价策略建议。支持 A/B 测试分析、ROAS 优化、竞品对比。

## 功能特性
- 📊 广告表现深度分析（CTR/CPC/ROAS）
- 🎨 创意素材优化建议
- 👥 受众定位优化
- 💰 智能出价策略
- 🧪 A/B 测试结果分析
- 🏆 竞品广告监测

## 使用方式
```bash
# 分析广告表现
/facebook-ad-optimizer-pro --analyze --campaign <ID> --days 7

# 创意优化建议
/facebook-ad-optimizer-pro --creative --ad <ID>

# 受众分析
/facebook-ad-optimizer-pro --audience --campaign <ID>

# ROAS 优化
/facebook-ad-optimizer-pro --roas --target 4.0 --campaign <ID>
```
