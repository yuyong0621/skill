# Book Recommend Tips

## 推荐原则

1. **了解读者** — 阅读水平、兴趣偏好、可用时间
2. **多样搭配** — 经典+畅销+小众，避免单一类型
3. **循序渐进** — 从入门到进阶，由浅入深
4. **附加价值** — 不只推荐书名，附上推荐理由

## 书单分类

| 场景          | 推荐策略                    |
|--------------|---------------------------|
| 入门某个领域    | 经典入门书 + 通俗读物        |
| 深入研究       | 体系化书单，有阅读顺序       |
| 碎片时间       | 短篇集、散文、轻阅读         |
| 提升认知       | 跨领域经典（心理/经济/哲学）  |
| 职场发展       | 技能书 + 思维书 + 传记       |

## 读书会设计

- **每次一本**：4周读完，每周讨论
- **准备问题**：5-8个开放式讨论题
- **活动设计**：破冰 → 分享 → 讨论 → 总结
- **延伸阅读**：推荐相关的文章、TED演讲

## 阅读方法

- **快速阅读**：目录→结论→感兴趣的章节
- **深度阅读**：做笔记、画思维导图、写读后感
- **主题阅读**：围绕一个话题读5-10本
- **费曼技巧**：读完后假装教给别人
