# 推荐合适的酒店

## 酒店类型与适用场景
- **商务酒店**：差旅、会议、短住，交通与网络便利
- **度假村 / 全包酒店**：家庭、蜜月、放松，含餐食与活动
- **民宿 / 公寓**：长住、多人、想体验当地生活
- **精品 / 设计酒店**：追求特色与体验，预算较宽松

## 选择要点
- 位置：距景点、交通枢纽、餐饮的距离
- 评价：评分、近期点评、卫生与安全反馈
- 设施：早餐、Wi‑Fi、停车、无障碍、儿童设施
- 政策：取消/改期、入住退房时间、加床与加人

## 特殊需求考量
- 亲子：儿童设施、加床、安全、安静程度
- 无障碍：电梯、无障碍房、卫生间与通道
- 饮食：含早、餐厅选择、过敏或宗教饮食需求

- [ ] 核对房型、人数与加床/加人费用
- [ ] 查看交通方式与周边便利度
