# Tool Configuration Reference

## Required Tools

All configurations must include the following tools:

| Tool ID | Name | Function |
|---------|------|----------|
| find-skills | Skill Discovery | Discover and install skills from OpenClaw marketplace |

| brave-search | Brave Search | Privacy-first web search |
