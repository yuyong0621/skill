# AI 保险经纪人 - Insurance Broker Skill

智能保险经纪人 OpenClaw Skill，提供需求分析、产品推荐、理赔协助等全方位保险服务。

## 安装

```bash
# 复制到 OpenClaw skills 目录
cp -r insurance-broker-skill ~/.openclaw/workspace/skills/insurance-broker

# 或通过 ClawHub 安装（待发布）
clawhub install insurance-broker
```

## 使用

与 OpenClaw 对话时提及保险相关需求即可自动激活：

- "我想买保险"
- "推荐保险产品"
- "分析我的保险需求"
- "重疾险和医疗险有什么区别？"

## 功能

- 🔍 保险需求分析 - 4大风险评估
- 📦 智能产品推荐 - 50+ 产品数据库
- 💡 保险知识问答 - 术语解释、投保建议
- 💰 保费快速计算 - 在线估算
