# Polymarket Market Search

Search and find specific prediction markets on Polymarket by keyword, category, or question text.

## Features

- **Keyword Search**: Find markets by keywords
- **Category Filter**: Filter by Politics, Sports, Crypto, Science
- **Status Filter**: Open, Resolved, Closed markets

## Pricing

- **Price**: 0.001 USDT per API call
- **Payment**: Integrated via SkillPay.me

This skill is integrated with SkillPay.me for automatic micropayments. Each call costs 0.001 USDT.
