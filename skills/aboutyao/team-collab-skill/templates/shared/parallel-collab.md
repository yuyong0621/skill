# 并行协作模板

## 并行执行模式

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│ 产品 Agent  │  │ 研发 Agent  │  │ 运营 Agent  │
│ 执行任务A   │  │ 执行任务B   │  │ 执行任务C   │
└──────┬──────┘  └──────┬──────┘  └──────┬──────┘
       │                │                │
       └────────────────┼────────────────┘
                        │
                        ▼
                  主 Agent 汇总
```

## 并行 Spawn

```javascript
spawnAgent('product', '任务A');
spawnAgent('dev', '任务B');
spawnAgent('ops', '任务C');
// 等待所有完成
```
