# 信息提取方法

## 从对话中提取

### 1. 关键词识别

#### 偏好信号
| 模式 | 示例 | 提取信息 |
|------|------|----------|
| 直接表达 | "我喜欢咖啡" | 偏好: 咖啡 |
| 比较选择 | "比起A，我更倾向B" | 偏好: B > A |
| 否定表达 | "我不吃香菜" | 忌讳: 香菜 |
| 频率词 | "我每天都喝" | 习惯: 高频行为 |

#### 情绪信号
| 模式 | 示例 | 提取信息 |
|------|------|----------|
| 情绪词 | "太开心了！" | 情绪: 快乐 |
| 感叹号 | "真的？！" | 情绪强度: 高 |
| 表情符号 | 😊😢😡 | 情绪类型 |
| 语气词 | "唉"、"哇" | 情绪基调 |

### 2. 模式识别

#### 时间模式
```
用户总在晚上10点后回复
→ 作息偏好: 夜猫子
→ 最佳互动时间: 22:00-24:00
```

#### 沟通模式
```
用户回复简短，不喜欢闲聊
→ 沟通风格: 直接、高效
→ 策略: 减少寒暄，直入主题
```

#### 决策模式
```
用户做决定前总问"你觉得呢？"
→ 决策风格: 咨询型
→ 需求: 提供分析和建议
```

### 3. 主题提取

#### 关注话题
```
多次提到"孩子教育"
→ 当前关注: 家庭教育
→ 可能角色: 父母
```

#### 专业领域
```
使用专业术语，讨论深度问题
→ 知识领域: [具体领域]
→ 专业程度: 高
```

## 从行为中提取

### 1. 交互行为

| 行为 | 可能含义 |
|------|----------|
| 快速回复 | 在线活跃/话题感兴趣 |
| 长时间沉默 | 忙碌/思考/犹豫 |
| 频繁提问 | 学习欲望强/不确定 |
| 分享链接 | 信任度高/寻求共鸣 |
| 修正AI | 高标准/明确需求 |

### 2. 选择行为

| 模式 | 提取信息 |
|------|----------|
| 总选方案A | 偏好: A类型方案 |
| 选择后后悔 | 决策困难/优柔寡断 |
| 快速决定 | 果断/直觉型 |
| 反复比较 | 审慎/分析型 |

### 3. 语言风格

| 特征 | 可能含义 |
|------|----------|
| 频繁使用"我们" | 集体意识强/家庭导向 |
| 频繁使用"我" | 自我意识强/独立 |
| 正式语言 | 职业素养/距离感 |
| 口语化 | 亲近感/放松 |
| 长句子 | 思维缜密/表达欲强 |
| 短句子 | 效率导向/简洁 |

## 深度分析

### 1. 价值观推断

从行为序列推断价值观：

```
行为1: 花时间帮陌生人 → 利他倾向
行为2: 为家人牺牲 → 家庭价值观
行为3: 拒绝不道德收益 → 正直
综合: 价值观 = 利他 + 家庭 + 正直
```

### 2. 需求层次分析

```
表达关注收入 → 可能在生理/安全需求层
表达关注认可 → 可能在尊重需求层
表达关注意义 → 可能在自我实现层
```

### 3. 认知风格判断

```
喜欢具体细节 → 具体思维者
喜欢抽象概念 → 抽象思维者
喜欢步骤指导 → 程序型学习者
喜欢整体图景 → 整体型学习者
```

## 信息验证

### 1. 交叉验证

```
来源A: "我喜欢运动"
来源B: 经常在周末早晨不在线
来源C: 提到过跑步装备
结论: 运动爱好者 (高置信度)
```

### 2. 时间验证

```
时间T1: "我在找工作"
时间T2 (2周后): "新工作很忙"
验证: 状态变化已确认
```

### 3. 用户确认

```
AI: "根据我们的交流，我觉得你更倾向于..."
用户: "对/不对/部分对"
更新: 根据反馈修正模型
```

## 敏感信息处理

### 需要授权的信息
- 健康状况
- 财务信息
- 家庭矛盾
- 心理状态
- 争议观点

### 处理原则
1. 不主动记录敏感信息
2. 如果用户主动分享，标记为敏感
3. 敏感信息不用于一般推理
4. 提供一键删除功能
