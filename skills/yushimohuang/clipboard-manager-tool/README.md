# Clipboard Manager Skill

Cross-platform clipboard history management for OpenClaw.

## Features

- ✅ Save clipboard history
- ✅ Search through past items
- ✅ Cross-platform (Windows/Mac/Linux)
- ✅ Automatic trimming (max 50 items)
- ✅ Quick restore

## Usage

### Save Clipboard
```bash
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh save
```

### Show Current
```bash
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh get
```

### Set Clipboard
```bash
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh set "Hello World"
```

### History
```bash
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh history
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh history 20
```

### Search
```bash
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh search "api"
```

### Clear
```bash
bash skills/clipboard-manager-1.0.0/scripts/clipboard.sh clear
```
