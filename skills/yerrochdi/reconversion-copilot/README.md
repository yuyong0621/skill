# Reconversion Copilot - OpenClaw Skill

Copilote IA de reconversion vers les metiers de la data. Accompagne les professionnels experimentes dans leur transition de carriere.

## Fonctionnalites

- **Diagnostic Express** : analyse de parcours, carte de competences, scores de compatibilite avec les roles data
- **Exploration Metiers** : fiches detaillees Data Analyst, AMOA Data, PO Data, BI Lead (salaires, marche, ROME)
- **Plan de Transition** : roadmap personnalisee 3/6/12 mois avec formations CPF, certifications, projets portfolio
- **Traduction Professionnelle** : reecriture CV, optimisation LinkedIn, lettres de motivation, pitch professionnel
- **Preparation Entretiens** : questions types, methode STAR, questions techniques adaptees, negociation salariale
- **Suivi & Motivation** : check-ins, tracking de progression, gestion du rejet, celebration des milestones

## Roles cibles

- Data Analyst
- AMOA Data
- PO Data (Product Owner Data)
- BI Lead

## Profils cibles

Professionnels avec 10+ ans d'experience en :
- Ressources Humaines (RH)
- SIRH (Systemes d'Information RH)
- AMOA (Assistance a Maitrise d'Ouvrage)
- IT / Gestion de projet
- Management fonctionnel

## Installation

```bash
clawhub install reconversion-copilot
```

## Utilisation

Le skill s'active automatiquement quand l'utilisateur parle de reconversion, transition data, diagnostic de competences, CV, entretien, etc.

### Exemples

- "Je suis responsable SIRH depuis 12 ans, je veux evoluer vers la data"
- "Fais-moi un diagnostic de mon profil pour la data"
- "Aide-moi a reecrire mon CV pour un poste de Data Analyst"
- "Prepare-moi pour un entretien AMOA Data"
- "On fait le point sur ma transition ?"
